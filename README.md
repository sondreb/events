# Events

Easy to use, easy to understand website that lists events for specific cities and regions. It's powered by an agent that runs once a day to fetch events from various sources and update the website.

It's deployed using GitHub Pages and built using Angular. It's not using a backend, it will pre-generate either all the HTML content, or a JSON database (or something similar) that is loaded pr. city/region and then rendered on the client side.

It should have all the standard and expected features of an events website, such as adding to calendars, sharing events, attempt to understand the event information and put it on a map, etc.

Obviously it should search many different sources for events, such as Facebook, Meetup, Eventbrite, general web search etc. There might be specific websites that has events for the region or city, such as government sites.

Supported locations in the first version:

- Bar, Montenegro
- Budva, Montenegro
- Kotor, Montenegro
- Tivat, Montenegro
- Podgorica, Montenegro
- Vennesla, Norway
